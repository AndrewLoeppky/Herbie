=============
herbie.wgrib2
=============

The ``_WGRIB2`` class is a wrapper for the wgrib2 executable. For a shortcut, you should import it with

.. code:: python

    from herbie.wgrib2 import wgrib2

where the ``wgrib2`` object imported is an instantiated class of ``_WGRIB2()``.
